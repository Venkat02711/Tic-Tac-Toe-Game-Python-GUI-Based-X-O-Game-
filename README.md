Tic Tac Toe Game – “Python GUI Based X-O Game”

The Tic Tac Toe game is a graphical user interface application developed using Python and Tkinter. It allows two players to play the classic X-O game on a 3×3 grid with real-time interaction and result display.

o Features:
Two-player gameplay, interactive buttons, automatic win and draw detection, and game reset functionality.

o Technologies Used:
Python, Tkinter

o Development Tools:
Python IDLE / VS Code
